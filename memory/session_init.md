# Checklist de início de sessão

Antes de realizar qualquer tarefa de edição de vídeo nesta sessão, ler nesta ordem:

1. [`docs/brand/gamer-hut-brandbook.md`](../docs/brand/gamer-hut-brandbook.md) — posicionamento, tom de voz, identidade visual.
2. [`docs/brand/style-guide-video.md`](../docs/brand/style-guide-video.md) — padrões de ritmo, legenda, transições (Fase 0).
3. [`memory/editing_rules_log.json`](editing_rules_log.json) — regras aprendidas com feedbacks anteriores.

Se alguma regra do log conflitar com o brandbook, priorizar o feedback mais recente do usuário e sinalizar a inconsistência.
